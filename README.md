### 프로젝트 목표
SNS 기능들을 구현.


### 기술 및 스택
* JAVA 11
* GRADLE
* MYSQL , REDIS
* EC2 , RDS ,S3, Route 53
* Jpa, QueryDsl
* Srping Security
* Thymleaf
* Jquery
* Jennkins

### 핵심기능
* 회원가입, 댓글, 팔로우, 좋아요
* 게시물 해쉬태그, 스크랩기능, 날씨 api 연동,
* Redis cache 적용
